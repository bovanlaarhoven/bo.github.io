Functions Reference

Available Functions

Bosploit provides a comprehensive set of functions for Minecraft automation and enhancement. All functions follow Roblox-style naming conventions and use the service-based architecture.

Core Services

game:GetService("Players")

Access player-related functions and properties.

local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer
print(LocalPlayer.Name)

game:GetService("Workspace")

Access world-related functions and the game environment.

local Workspace = game:GetService("Workspace")
local blocks = Workspace:GetChildren()

game:GetService("ReplicatedStorage")

Access shared game data and remote functions.

local ReplicatedStorage = game:GetService("ReplicatedStorage")
local remotes = ReplicatedStorage:GetChildren()

Player Functions

LocalPlayer:Teleport(Vector3)

Teleports the player to the specified coordinates.

local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer
LocalPlayer:Teleport(Vector3.new(100, 64, 200))

LocalPlayer.Character.Humanoid.Health

Gets or sets the player's health.

local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer
local humanoid = LocalPlayer.Character.Humanoid
humanoid.Health = 20 -- Set full health

LocalPlayer.Character.HumanoidRootPart.Position

Gets or sets the player's position.

local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer
local position = LocalPlayer.Character.HumanoidRootPart.Position
print("Player position:", position)

Workspace Functions

Workspace:GetBlockAt(Vector3)

Returns the block at the specified coordinates.

local Workspace = game:GetService("Workspace")
local block = Workspace:GetBlockAt(Vector3.new(100, 64, 200))
print("Block type:", block.Name)

Workspace:SetBlock(Vector3, string)

Places a block at the specified coordinates.

local Workspace = game:GetService("Workspace")
Workspace:SetBlock(Vector3.new(100, 64, 200), "Stone")

Workspace:CreateExplosion(Vector3, number)

Creates an explosion at the specified coordinates with the given power.

local Workspace = game:GetService("Workspace")
Workspace:CreateExplosion(Vector3.new(100, 64, 200), 4)

Combat Functions

LocalPlayer:Attack(Entity)

Attacks the specified entity.

local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer
local target = Workspace:FindFirstChild("Zombie")
if target then
    LocalPlayer:Attack(target)
end

LocalPlayer:LookAt(Vector3)

Aims the player's crosshair at the specified coordinates.

local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer
LocalPlayer:LookAt(Vector3.new(100, 64, 200))

LocalPlayer:Shoot()

Shoots with the current weapon (bow, crossbow, etc.).

local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer
LocalPlayer:LookAt(target.Position)
LocalPlayer:Shoot()

Utility Functions

wait(number)

Pauses script execution for the specified number of seconds.

wait(2.5) -- Wait 2.5 seconds

print(...)

Prints messages to the console.

print("Script started successfully!")
print("Position:", LocalPlayer.Character.HumanoidRootPart.Position)

game:GetService("RunService").Heartbeat

Connects to the game's frame update event.

local RunService = game:GetService("RunService")
local connection = RunService.Heartbeat:Connect(function()
    -- This runs every frame
    print("Frame update")
end)

Vector3.new(x, y, z)

Creates a new Vector3 object for 3D coordinates.

local position = Vector3.new(100, 64, 200)
local offset = Vector3.new(10, 0, 10)
local newPosition = position + offset

Advanced Services

game:GetService("TweenService")

Create smooth animations and transitions.

local TweenService = game:GetService("TweenService")
local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer

local tweenInfo = TweenInfo.new(2, Enum.EasingStyle.Quad)
local tween = TweenService:Create(
    LocalPlayer.Character.HumanoidRootPart,
    tweenInfo,
    {Position = Vector3.new(100, 64, 200)}
)
tween:Play()

game:GetService("UserInputService")

Handle user input events.

local UserInputService = game:GetService("UserInputService")

UserInputService.InputBegan:Connect(function(input, gameProcessed)
    if input.KeyCode == Enum.KeyCode.E then
        print("E key pressed!")
    end
end)
