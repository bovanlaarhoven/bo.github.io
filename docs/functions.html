Custom Functions

üë§ Player Functions

game.Players.LocalPlayer

Access the local player object for position, health, and other properties.

local player = game.Players.LocalPlayer
print("Player name: " .. player.Name)
print("Health: " .. player.Character.Humanoid.Health)

setWalkSpeed(speed)

Modify player movement speed with custom values.

setWalkSpeed(50) -- Set speed to 50
setWalkSpeed(16) -- Reset to default speed

setJumpPower(power)

Adjust player jump height and power.

setJumpPower(100) -- High jump
setJumpPower(50)  -- Default jump

üåç World Functions

getBlockAt(x, y, z)

Retrieve block information at specific coordinates.

local block = getBlockAt(100, 64, 200)
print("Block type: " .. block.type)
print("Block hardness: " .. block.hardness)

setBlock(x, y, z, blockType)

Place blocks at specified coordinates (creative mode).

setBlock(100, 65, 200, "diamond_block")
setBlock(101, 65, 200, "gold_block")

mineBlock(x, y, z)

Automatically mine blocks at given coordinates.

mineBlock(player.x, player.y - 1, player.z)
-- Mines the block below the player

üîß Utility Functions

wait(seconds)

Pause script execution for specified duration.

wait(2)      -- Wait 2 seconds
wait(0.5)    -- Wait half a second

print(message)

Output messages to the console for debugging.

print("Script started!")
print("Current position: " .. player.x .. ", " .. player.y .. ", " .. player.z)

getDistance(x1, y1, z1, x2, y2, z2)

Calculate distance between two points in 3D space.

local dist = getDistance(0, 0, 0, 100, 0, 100)
print("Distance: " .. dist)

üìã Function Example

Here's a comprehensive example combining multiple functions:

-- Advanced auto-farming script
local player = game.Players.LocalPlayer
local farming = false

function autoFarm()
    farming = true
    setWalkSpeed(25) -- Increase speed for efficiency
    
    while farming do
        -- Scan 5x5 area around player
        for x = -2, 2 do
            for z = -2, 2 do
                local blockX = player.x + x
                local blockZ = player.z + z
                local blockY = player.y - 1
                
                local block = getBlockAt(blockX, blockY, blockZ)
                
                -- Check for valuable blocks
                if block.type == "diamond_ore" or 
                   block.type == "gold_ore" or 
                   block.type == "iron_ore" then
                    
                    print("Found " .. block.type .. " at " .. blockX .. ", " .. blockY .. ", " .. blockZ)
                    mineBlock(blockX, blockY, blockZ)
                    wait(0.5) -- Brief pause between mining
                end
            end
        end
        
        wait(1) -- Scan every second
    end
end

function stopFarming()
    farming = false
    setWalkSpeed(16) -- Reset to normal speed
    print("Auto-farming stopped")
end

-- Start the auto-farm
print("Starting auto-farm...")
autoFarm()
